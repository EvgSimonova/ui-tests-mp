# Описание #
 
Тесты для веб-прилоежения marc-project


### Как пользоваться? ###

* Установить maven
* Скачать репозиторий
* Перейти в папку с репозиторием
* Запустить команду mvn clean install (mvn -Dtest=TestCircle test)
