# Описание #
 
Тесты для веб-прилоежения marc-project


### Как пользоваться? ###

* Установить maven
* Установить selenium firefox webdriver
* Установить geb
* Скачать репозиторий
* Перейти в папку с репозиторием
* Запустить команду mvn clean install
